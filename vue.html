vue

  • 组件文件组织结构模板、样式及脚本分离
    反模式示例(vue官网推荐的单文件组件):
      <template></template>
  <style></style>
  <script></script>
    最佳实践(为每个组件创建一个文件夹,文件夹名称为组件名,文件夹内分别创建组件及模板、样式、脚本文件):
      <template src="./ComponentName.template.html"></template>
  <style src="./ComponentName.style.css"></style>
  <script src="./ComponentName.script.js"></script>
  • 禁止组件内部定义全局样式(需要用到全局样式请定义在外部文件并使用@import进来或者在入门文件内import)
    反模式示例:
      <template>
    <div class="item">
      <div>
        <span>hello world</span>
      </div>
    </div>
  </template>
  <style>
    .item span{
      color:red;
    }
  </style>
  <script></script>
    最佳实践:
      <template>
    <div class="item">
      <div>
        <span>hello world</span>
      </div>
    </div>
  </template>
  <style scoped>
    .item >>> span{
      color:red;
    }
  </style>
  <script></script>
  • 禁止单文件代码行数超过500(建议拆分成更小的功能单元或组件单元)
  • 为每个vue组件添加props描述
    反模式示例:
      export default {
      name: 'custom-person',
      props:['name','age']
  }
    最佳实践:
      export default {
      name: 'custom-person',
      props: {
        name: {
          type: String,
          required: true,
          default: '',
          description: '名字'
        },
        age: {
          type: Number,
          required: true,
          default: 20,
          description: '年龄'
        }
      }
  }
  • 使用计算属性来替代模板内复杂的插值表达
    反模式示例:
      <template>
      \{\{this.array.map(item=>return item*2)\}\}
  </template>
    最佳实践:
    <template>
    \{\{this.arrayDouble\}\}
</template>
<script>
    export default {
        data(){
            array:[1,2,3]
        },
        computed(){
            arrayDouble:function(){
                return this.array.map(item=>item*2);
            }
        }
    }
</script>
  • 组件销毁时清除组件内的定时器及手动添加的事件处理
    反模式示例:
      export default {
    mounted() {
        document.body.addEventListener('mousemove',e=>{
            ...
        });
    },
      methods: {
          doSomething() {
              this.timer = setTimeout(fn,time);
          }
      }
  }
    最佳实践:
      export default {
    mounted() {
        document.body.addEventListener('mousemove',this.onMousemove);
    },
    methods:{
      /**
       * @desc 做某件事
       * @return {undefined} 无返回值
       */
        doSomething(){
            this.timer = setTimeout(fn,time);
        },
        /**
         * @desc 鼠标移动时
         */
        onMousemove(e){
            ...
        }
    },
    destroyed(){
        clearTimeout(this.timer);
        document.body.removeEventListener('mousemove',this.onMousemove);
    }
  }

      import axios from 'axios';
  import dynamics from  'dynamics.js'; 
  let dom = this.refs.dom;
  let {width,height} = dom.getBoundingClientRect();
  let childs = dom.querySelectorAll('.child');
  dom.classList.add('className');
  dom.classList.remove('className');
  dom.classList.toggle('className');
  axios({
      url,
      method
  })
  .then(res=>{
      ...
  })
  .catch(err=>{
      ...
  });
  dynamics.animate(dom,{
      translateX: 100,
      translateY: 50
  },{
      type: dynamics.linear,
      duration: 1000
  });
