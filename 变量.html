变量

变量

  • 禁止定义全局变量(适当允许定义全局类)
    反模式示例:
      window.person = new Person(...);
    最佳实践(使用IIFE来创建一个作用域用于隔离全局):
      let person = (function(){
    let person = new Person;
    return person;
  }());
  • 禁止使用var关键字来定义变量
    反模式示例:
      var string = '';
  var date = new Date();
    最佳实践(使用let或const关键字声明变量):
      let string = '';
  const date = new Date();
  • 禁止对象属性写在同一行
    反模式示例:
      let obj = {a: 1, b: 2, c: 3};
    最佳实践：
      let obj = {
    a: 1,
    b: 2, 
    c: 3
  };
  • 禁止使用关键字作用对象的属性名
    反模式示例:
      let obj = {
    delete(){}
  };
    最佳实践：
      let obj = {
    remove(){}
  };
  • 总是在所在作用域顶部定义
    反模式示例:
      function defineVar(){
     let v1 = 123;
     doSomething();
     let v2 = 456;
  };
    最佳实践：
      function defineVar(){
     let v1 = 123;
     let v2 = 456;
     doSomething();
  };
  • 块级作用域内使用let关键字
    反模式示例:
      function defineVar(){
      let v1 = 123;
      for(var i=0;i<v1;i++){
         doSomething(i);
      }    
  };
    最佳实践：
      function defineVar(){
      let v1 = 123;
      for(let i=0;i<v1;i++){
         doSomething(i);
      }    
  };
  • 通过模块导入的解构使用const关键字 反模式示例:
      import * as someModule from 'someModule';
  let {f1, f2} = someModule;
    最佳实践：
      import * as someModule from 'someModule';
  const {f1, f2} = someModule;
  • 不使用的变量建议注释或删除
