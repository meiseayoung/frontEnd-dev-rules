定时器

定时器

  • 禁止使用setInterval(因为无法保证fn的执行时长，如果fn执行太长时间会导致函数被堆积起来)
    反模式示例:
      function fn(){
      ...;
  }
  setInterval(fn,1000);
    最佳实践:
      function fn(){
    ...;
    if(condition){
      setTimeout(fn,1000);
    }
  }
  • 保留定时器的使用句柄以供将来清除
    反模式示例:
      setTimeout(fn,1000);
    最佳实践:
      let timer = setTimeout(fn,1000);
