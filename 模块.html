模块

模块导入

  • 一个export只导出一个功能模块
    反模式示例:

      function show(){

  };
  function hide(){

  };
  export default {
    show,
    hide
  }

    最佳实践:

      export function show(){

  }
  export function hide(){

  }
  • 禁止重复导入模块
    反模式示例:
      import { merge } from 'module';
  import something from 'another-module';
  import { find } from 'module';
    最佳实践:
      import { merge,find } from 'module';
  import something from 'another-module';
