隐式类型转换

隐式类型转换

  • 使用===和!==而不是==和!=(避免隐式类型转换带来的问题)
    反模式示例:
      let string = '';
  let string2 = false;
  string == string2 && doSomething();
  string != string2 && doSomething2();
    最佳实践:
      let string = '';
  let string2 = false;
  string === string2 && doSomething();
  string !== string2 && doSomething2();
    或者使用lodash库的isEqual方法作比较
      import isEqual from 'lodash/isEqual';
  let string = '';
  let string2 = false;
  isEqual(string,string2) && doSomething();
  !isEqual(string,string2) && doSomething2();
    eslint规则: 
      "eqeqeq":"warn"
  • 如果需要做比较需要强制转换成布尔类型，转换遵循以下规则
    1. 对象=>true
    2. undefined => false
    3. null => false
    4. boolean => boolean
    5. number => 0或NaN => false 其它 => true
    6. string => string.trim() === '' => false 其它 => true
